🎉 Decap CMS 로컬 테스트

✅ Decap CMS가 성공적으로 설치되었습니다!

로컬에서 Decap CMS를 테스트하려면 아래 링크를 사용하세요:

📝 사용 가능한 컬렉션:

  • 일기 (Journal) - 개인 일기나 생각 정리
  • 독서 (Book) - 책 리뷰나 독서 기록
  • 영화 (Movie) - 영화 리뷰나 영화 관련 글
  • 갤러리 (Gallery) - 이미지 갤러리
  • 초안 (Draft) - 아직 발행하지 않은 초안 글

🔗 접속 링크:

실제 CMS (Git 연동) 메인 사이트

💡 사용 팁:

  • 실제 CMS: GitHub 계정으로 로그인하여 실제 콘텐츠 작성
  • 로컬 테스트: 현재 페이지에서 UI 확인
  • 이미지 업로드: 드래그 앤 드롭으로 쉽게 업로드
  • 마크다운: 실시간 미리보기 지원

🚀 다음 단계:

  1. 위의 "실제 CMS" 버튼을 클릭
  2. GitHub 계정으로 로그인
  3. 원하는 컬렉션에서 새 콘텐츠 작성
  4. 에디토리얼 워크플로우를 통해 콘텐츠 관리
